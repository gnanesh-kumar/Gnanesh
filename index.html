Projects

Drug Delivery Robot Using IoT:

This project aims to design a low- cost drug delivery robot for hospital and improve the efficiency and accuracy of medication delivery in healthcare through automation technique. The drug delivery line following robot using IoT (internet of things). The IoT technology enables the robot to communicate with internet and other device. This robot is designed to follow a predefined path on the floor and deliver medication to patient’s rooms. The system is built using a board and various IoT components such as sensors and wireless communication protocol ls that enable the robot to be controlled remotely. The drug delivery line following robot has several advantages over traditional delivery methods. For one, it is more accurate and reliable, ensuring that medication is delivered to the correct patient and location. It is also more efficient, as the robot can deliver medication to multiple patients at once, reducing the need for human intervention.
